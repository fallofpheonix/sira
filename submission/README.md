# SIRA Submission Bundle

This folder collects the minimum submission-facing materials for the SIRA evaluation repository.

## Included

- `PROJECT_SUMMARY.md`: maps the repository to the requested deliverables.
- `RUN_INSTRUCTIONS.md`: validated commands for environment setup and a fast end-to-end demo run.
- `SUBMISSION_CHECKLIST.md`: final handoff checklist aligned with `docs/submission_guidelines.md`.
- `artifacts/`: copied outputs from the validated submission demo run.

## Source Documents

Authoritative project documents remain in the repository root and `docs/`:

- `README.md`
- `docs/problemstatement.md`
- `docs/proposal.md`
- `docs/constraints.md`
- `docs/submission_guidelines.md`

Use this bundle as the concise pointer set when sharing the repository link with reviewers.
