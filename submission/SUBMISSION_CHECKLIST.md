# Submission Checklist

Submit the following by email to `human-ai@cern.ch` with the required evaluation subject line:

1. Current CV or resume.
2. Repository link for this project.
3. A short note pointing reviewers to:
   - `submission/README.md`
   - `submission/RUN_INSTRUCTIONS.md`
   - `README.md`

## Repository Readiness

- [x] Installation steps documented.
- [x] Simulation, training, and symbolic-discovery pipeline implemented.
- [x] Verified smoke run command included.
- [x] Verified pytest command included.
- [x] Submission-specific demo config included.

## Optional Attachments

- PDF report or summary note, if requested separately.
- Exported plots from `results/` if you want to highlight qualitative outputs.
