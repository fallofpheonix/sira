# Submission Guidelines & Constraints: SIRA

## What to Build
1. **Simulation Model**: Starting with the stochastic equations, build a simulation of an epidemic based on the SIR model.
2. **Predictive ML Model**: Train a Machine Learning model to predict the mean counts of Susceptible (S), Infected (I), and Removed (R) populations over time.
3. **Symbolic Approximation**: Use auto-differentiation and symbolic methods to approximate the continuous functions `S(t)`, `I(t)`, and `R(t)`.

## What to Submit to Mentor
Applicants must submit the following via email to **human-ai@cern.ch** with **"Evaluation Test: SIRA"** (or the project title limit) in the subject line:
- A current CV/Resume.
- A GitHub repository link containing the code for the tasks listed above.
- Ensure clear documentation on how to run your simulations and models.

## Constraints
1. **Language/Frameworks**: Python, with PyTorch or TensorFlow.
2. **Mathematical Prerequisites**: The solution requires a solid understanding of upper-level mathematics, specifically differential equations and linear algebra.
3. **Communication**: Do NOT contact mentors directly by email.
